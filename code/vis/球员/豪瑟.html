球员介绍

        山姆-豪瑟(英文名:Sam Hauser，1997年12月8日出生)家乡位于美国威斯康星州格林贝 ，美国职业篮球运动员，身高203m，体重99kg，司职前锋。

        大学时期就读于马奎特，大一赛季，豪瑟场均可以得到 8.8分、5 个篮板和 1.3 次助攻，三分命中率达到 45.3%，同时在上场时间和防守篮板方面领先全队。大二赛季，场均得到 14.9 分7.2 个篮板和 2.4次助攻，并入选了大东区第二阵容。随后，山姆-豪瑟转学到弗吉尼亚大学，2020年11月25在弗吉尼亚的处子秀中，他在89-54战胜陶森的比赛中得到19分。2021年3月25日，豪瑟宣布他将参加2021年的NBA选秀。

        山姆 豪瑟身高臂展灵活性不错，但爆发力平平，有一定的持球能力，善于转换和挡拆进攻，既可以作为挡拆的发起人，也可以做终结点，终结能力不错，有一定的组织能力，但失误较多。缺点是防守不好，移速度偏，协防意识不好。

        在2021年 NBA选秀中落选后，豪瑟于2021年8月 13日与波士顿凯尔特人队签订了一份双向合同。
